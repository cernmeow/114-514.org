# 114-514.org

Net Work ?

A lightweight browser-based HTTP reachability test.

Measures HTTP reachability from the browser itself rather than ICMP echo latency.

Unlike traditional ping tools, results reflect the ability of a browser to reach a website through DNS, TLS and HTTP.
## Features

- Browser-only
- No installation
- HTTP RTT measurement
- Jitter measurement
- Timeout loss estimation
- Mobile friendly
